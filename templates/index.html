API Documentation

Capture a screenshot of any website with customizable quality and optional Cloudflare bypass.

Parameter Type Description
urlrequired string The website URL to capture
qualityoptional string low | hd | fhd | wqhd (default: hd)
bypassoptional boolean Enable Cloudflare bypass (default: false)

Example Request:

curl -X GET "http://your-api:3674/web/ss?url=https://example.com&quality=hd&bypass=false"

Example Response:

Retrieve a captured screenshot by its file ID. Files expire after 60 seconds.

Parameter Type Description
file_idrequired string The unique file identifier returned from /web/ss

curl -X GET "http://your-api:3674/file/abc123" --output screenshot.png
